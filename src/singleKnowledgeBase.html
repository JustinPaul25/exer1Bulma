How to Install WordPress Theme?

At first please make sure that you have unpacked the whole package shown in the last point. This is very important! Tou can complete installation in two different ways.

  WordPress Theme Manager  - Please go to Appearance > Themes > Add new > Upload, select zipped theme folder called theme zip and press Install Now Button The WordPress will do the rest for you.
  FTP Upload  - upload non-zipped theme folder betheme to /wp-content/themes/ folder in your WordPress intallation folder on your server.

In the Second Part of this guide we havve describe how to update juster theme. Please make sure that you didnt make any changes in the files. If you have made changes and you don'y want to lose them, the best way is to use one of freeware tools for comparing files. ex. Beyond Compare (we are not the authors of this tool so please don't ask us how to use it. Answers to how to use it can be found on their website). After comparing these files you can upload on server only the files that hasn't been changed.

If you haven't change any theme files, all you need to do is update your version of theme by downloading the latest package from themeforest. unpacking files ang uploading new files on the server while replacing the old files. Simply overwrite them.


Note: All your data in database are safe because after updating theme file doesn't change anything in the database. Please however always make a backup of your lines & database.

IMPORTANT!!!  After theme installation. at the top you have box with information abaout required and optional plugins. To install them and activate, please navigate to Appearance > install Plugins section and install only NECESSARY PLUGINS$nbsp . We include plugins like Slider Revolution(required). Contact Form 7(required) and Visual Composer(required).
